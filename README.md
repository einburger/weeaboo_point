
### About ###

* This is a VN engine that reads from a movie-like script.

## Example script ##
```
at station_a.jpg

place alice 50
place preppy 50
place swimsuit 50

> *audible frustration*

fadein preppy 0.05
wait preppy 0.5

> Hello.

move preppy 20 5
sync
move swimsuit 53 5
sync
fadein swimsuit 0.05
sync
fadein alice 0.02 

wait preppy 0.5 

> HUUAAH! Die mudcrab!

move swimsuit 20 20
```
